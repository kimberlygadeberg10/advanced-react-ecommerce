# Advanced React E-Commerce App

This project is an advanced e-commerce web application built with React, TypeScript, Vite, React Query, and Redux Toolkit. It uses the FakeStoreAPI to display store products and categories and is being developed as a module project focused on modern React concepts.

## Features Completed So Far

- React + TypeScript app created with Vite
- Routing set up with React Router
- React Query provider configured
- Redux store configured
- Product data fetched from FakeStoreAPI
- Category data fetched dynamically from FakeStoreAPI
- Product filtering by category
- Reusable product card component
- Image fallback for broken product image URLs
- Responsive product card layout improvements

## Project Structure

- `src/api` for API request functions
- `src/components` for reusable UI pieces
- `src/pages` for routed pages
- `src/store` for Redux store setup
- `src/types` for shared TypeScript types

## Technologies Used

- React
- TypeScript
- Vite
- React Router DOM
- React Query
- Redux Toolkit
- FakeStoreAPI

## API Endpoints Used

- `https://fakestoreapi.com/products`
- `https://fakestoreapi.com/products/categories`
- `https://fakestoreapi.com/products/category/{category}`
